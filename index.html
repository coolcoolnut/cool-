☁️ GitHub Gist 同步

用 GitHub Gist 免费同步数据到云端，任何设备都能读取。
需要一个 GitHub 账号和 Personal Access Token。

💡 获取 Token 步骤：
1. 打开 github.com → 右上角头像 → Settings
2. 左侧最底部 Developer settings → Personal access tokens → Tokens (classic)
3. Generate new token → 勾选 gist 权限 → 复制 token

⚠️ 出现 401 错误：确认 token 有 gist 权限，且未过期。手机和电脑填写相同的 Token 和 Gist ID 即可同步。
